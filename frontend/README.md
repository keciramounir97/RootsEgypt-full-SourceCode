# RootsEgypt Frontend

A Roots Egypt heritage platform focused on family history, archives, visual collections, and historical research across Egypt.

## Project Structure

- `frontend/` - React + Vite application
- `photos/` - source imagery assets used during integration

## Tech Stack

- React 19
- Vite 6
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand
- TanStack Query
- React Router

## Configuration

- `frontend/.env` - environment variables
- `frontend/vite.config.js` - Vite configuration
- `frontend/tailwind.config.js` - styling configuration
- `frontend/tsconfig.json` - TypeScript configuration

## Development

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
cd frontend
npm run build
```

The production output is generated in `frontend/dist/`.
